Free Time Finder

チーム全員の空き時間を検索・共有するBot

概要

複数人のGoogleカレンダーを参照して、全員が空いている時間を自動検索するBotです。MTG調整の手間を大幅に削減できます。

LINE版

LINEで 。 を送信するだけで空き時間を表示

Slack版

Slackで /cal コマンドを実行してチャンネルに共有

機能

機能 説明
全員の空き時間検索 複数のGoogleカレンダーを参照し、全員が空いている時間帯を抽出
営業時間フィルタ 10:00〜19:00の間で空き時間を検索（土日除外）
最小スロット 1時間以上の空きのみ表示
週間表示 今日から7日間の空き時間を一覧表示

使い方

LINE版

コマンド 説明
。 今週の空き時間を表示
空き 今週の空き時間を表示
ヘルプ 使い方を表示

Slack版

コマンド 説明
/cal 今週の空き時間をチャンネルに共有
/cal help ヘルプを表示

出力例

【全員の空き時間】
2/3(月) 10:00-12:00, 15:00-17:00
2/4(火) 13:00-19:00
2/5(水) 10:00-11:00, 16:00-19:00
2/6(木) 10:00-19:00
2/7(金) 14:00-19:00

セキュリティ 安全

読み取り専用アクセス
このBotはカレンダーの「閲覧のみ」が可能です。予定の追加・編集・削除はできません。
✓
署名検証 - LINE/Slackからのリクエストを暗号署名で検証
✓
シークレット暗号化 - APIキーはCloudflareで暗号化保存
✓
HTTPS通信 - すべての通信は暗号化
✓
最小権限 - calendar.readonly スコープのみ使用

技術構成

Cloudflare Workers TypeScript Google Calendar API LINE Messaging API Slack API

アーキテクチャ

コンポーネント 役割
Cloudflare Workers サーバーレス実行環境（エッジコンピューティング）
Google Calendar API カレンダーの予定取得（サービスアカウント認証）
LINE/Slack API メッセージング（Webhook受信・返信）

環境変数（シークレット）

LINE版

変数名 説明
LINE_CHANNEL_SECRET LINEチャネルシークレット
LINE_CHANNEL_ACCESS_TOKEN LINEチャネルアクセストークン
GOOGLE_SERVICE_ACCOUNT_EMAIL サービスアカウントのメールアドレス
GOOGLE_SERVICE_ACCOUNT_PRIVATE_KEY サービスアカウントの秘密鍵
GOOGLE_CALENDAR_IDS カレンダーID（カンマ区切りで複数指定）
